What is K-Nearest Neighbors (KNN)?

The K-Nearest Neighbors (KNN) algorithm is a non-parametric, instance-based learning method used for both classification and regression. It predicts the output class of a new data point based on the majority class among its closest data points—or "neighbors"—in the training dataset.

In EduVision’s context, KNN classifies students’ peer influence as Positive, Neutral, or Negative, reflecting whether their social circles contribute to supportive, neutral, or harmful academic behaviors.

What I Did & How I Did It

The model was trained using the KNN_PeerInfluence_RealFeatures dataset to study how social and academic attributes interact to shape peer influence patterns among students.

FEATURE_COLS = ['ParentalInvolvement', 'SchoolType', 'MotivationLevel']
TARGET_COL = 'PeerInfluence'

Each categorical feature was encoded into numerical form and standardized to ensure fair distance comparisons between data points. The optimal K value was determined experimentally for balanced accuracy.

  • Normalized all input features to prevent distance bias.
  • Tested multiple K values to find the best-performing configuration.
  • Evaluated using accuracy, precision, recall, and F1-score metrics.
