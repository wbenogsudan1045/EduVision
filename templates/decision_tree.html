What is a Decision Tree?

A Decision Tree is a supervised learning algorithm that models decisions through a flowchart-like structure. Each branch represents a test on a feature, while each leaf node corresponds to a class prediction.

In EduVision’s system, the Decision Tree is used to categorize students into strands such as ABM, GAS, HUMSS, and STEM, based on their aptitude performance across several reasoning categories.

What I Did & How I Did It

The model was trained using the Data_High_Separation_Strands dataset, which contains aptitude scores highly correlated with strand placements. Each attribute contributes to determining the most likely academic path for the student.

FEATURE_COLS = ['Numerical Aptitude', 'Spatial Aptitude', 'Perceptual Aptitude', 'Abstract Reasoning', 'Verbal Reasoning']
TARGET_COL = 'StrandCategory'

The Decision Tree splits data based on thresholds that maximize information gain. Hyperparameter tuning (depth, split criteria) ensured high classification accuracy and reduced overfitting.

  • Trained and validated on multiple folds for balanced accuracy.
  • Visualized tree splits to interpret feature importance.
  • Evaluated using precision, recall, F1-score, and confusion matrix.

This indicates that strand boundaries for GAS are more fluid, but the model remains highly useful for guiding academic recommendations and student counseling.
