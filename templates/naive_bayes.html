What is Naive Bayes?

Naive Bayes is a simple yet powerful classification algorithm based on Bayes’ Theorem. It calculates the probability that a given input belongs to a specific class, assuming that each feature is independent from others.

It’s widely used for tasks like spam detection, sentiment analysis, and academic performance prediction — where outcomes are categorized rather than continuous.

Study Hours per Week – Weekly time invested in studying.
Attendance Rate – Student participation and presence.
Final Exam Score – Overall exam performance indicator.

What I Did & How I Did It

Using the Student Performance Dataset, I trained a Gaussian Naive Bayes model to predict whether a student would Pass or Fail based on their study habits and exam results.

FEATURE_COLS = ['Study_Hours_per_Week', 'Attendance_Rate', 'Final_Exam_Score']
TARGET_COL = 'Pass_Fail'

Steps Taken:

  • Preprocessed and cleaned dataset to remove missing values.
  • Split data into training and test sets (80/20 ratio).
  • Trained model using GaussianNB() from scikit-learn.
  • Evaluated model with accuracy, precision, recall, and F1 score metrics.
