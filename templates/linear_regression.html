Linear Regression Model

Explore how student performance can be accurately predicted through attendance rate, study hours, and previous grades.

What is Linear Regression?

Linear Regression is a fundamental supervised machine learning algorithm used for predicting continuous values. It models the relationship between input variables (features) and an output variable (target) by fitting a linear equation to observed data.

In this project, the goal was to predict a student’s Final Grade based on three academic performance indicators:

Attendance Rate – Overall class participation percentage.
Study Hours per Week – Number of hours dedicated to study.
Previous Grade – Average score from past assessments.

What I Did & How I Did It

Using the Student Performance Dataset from Kaggle, a Linear Regression model was trained to analyze how attendance, study time, and past grades collectively influence a student’s final outcome.

The model configuration was as follows:

FEATURE_COLS = ['AttendanceRate', 'StudyHoursPerWeek', 'PreviousGrade']
TARGET_COL = 'FinalGrade'

The model was trained using the Ordinary Least Squares (OLS) optimization method, minimizing prediction error across all observations. The coefficients represent the contribution of each feature:

Intercept: 80.2777
AttendanceRate: +0.9230
StudyHoursPerWeek: −0.4658
PreviousGrade: +9.0595

Interpretation:

  • Higher attendance and stronger previous grades correlate with better final results.
  • A slight negative correlation with study hours may indicate inefficient study habits or overcompensation.

Model Evaluation

The model’s performance was evaluated using standard regression metrics, which measure accuracy and consistency of predictions.

Mean Absolute Error (MAE): 0.3885
Mean Squared Error (MSE): 0.2000
R² Score: 0.9964

These evaluation results indicate a highly accurate fit between predicted and actual grades, confirming the model’s robustness in predicting student performance.
