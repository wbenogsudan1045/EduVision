What is an Artificial Neural Network (ANN)?

An Artificial Neural Network is a computational model inspired by the human brain. It uses multiple layers of neurons that process information hierarchically, enabling the system to detect non-linear relationships and subtle trends in data.

In EduVision, this ANN helps predict whether students are likely to be high achievers, supporting academic planning and personalized learning strategies.

What I Did & How I Did It

The model was developed using the StudentsPerformance_with_high_achiever dataset. Three primary features were utilized:

Math Score — Measures numerical and analytical reasoning ability.
Reading Score — Evaluates comprehension and interpretation skills.
Writing Score — Assesses composition and critical expression capabilities.

Model Configuration:

FEATURES = ["math score", "reading score", "writing score"]
TARGET_COL = "high_achiever"

The network architecture includes multiple dense layers with ReLU activation and dropout regularization. It was trained using backpropagation to minimize categorical cross-entropy loss.

  • Normalized data to ensure consistent input scaling.
  • Used Adam optimizer for efficient convergence.
  • Evaluated using accuracy and loss metrics on training and validation sets.

Model Evaluation & Interpretation

The ANN model’s performance on training and validation data is summarized below:

Final Training Accuracy: 0.9962
Final Validation Accuracy: 0.9400
Final Training Loss: 0.1172
Final Validation Loss: 0.5016

Interpretation:
The model achieved an exceptional 99.6% accuracy on training and maintained a robust 94% validation accuracy, indicating strong learning capability with minimal overfitting.

Slightly higher validation loss reflects real-world variability, which is expected in educational datasets.
Overall, the ANN demonstrates reliable generalization, effectively identifying high achievers based on academic scores — a valuable decision-support feature for educators in personalized learning programs.
