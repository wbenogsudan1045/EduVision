What is Support Vector Machine (SVM)?

Support Vector Machine (SVM) is a supervised learning algorithm designed for classification tasks. It separates data into distinct classes by identifying the optimal hyperplane that best divides the categories with the maximum margin.

In EduVision’s context, the SVM model categorizes students into Low, Medium, or High Motivation levels. This helps educators identify students requiring emotional or academic support early on.

What I Did & How I Did It

The model was developed using the SVM Motivation Dataset to uncover patterns between academic behavior and motivation. Each feature was scaled and preprocessed to enhance classification accuracy.

FEATURE_COLS = ['Attendance', 'PreviousScores', 'Parental_Involvement']
TARGET_COL = 'MotivationLevel'

The dataset was divided into training and test subsets. Using a Radial Basis Function (RBF) kernel, the model effectively separated motivation categories based on non-linear relationships.

  • Trained with grid search optimization for best parameters.
  • Evaluated using accuracy, precision, recall, and F1-score metrics.
  • Validated through confusion matrix and classification report for consistency.
